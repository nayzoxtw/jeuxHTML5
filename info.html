Explication des Jeux

Bienvenue sur la Page d'Explication des Jeux

Guess the Number

  • Objectif: Deviner le bon nombre.
  • Contrôles: Utilise les flèches pour entrer tes suppositions et démarrer le jeu.

Clique sur les Formes

  • Objectif: Clique sur les formes avant qu'elles ne disparaissent.
  • Contrôles: Clique sur les formes avec la souris.

Démineur

  • Objectif: Dévoiler toutes les cases sans toucher de mine.
  • Contrôles: Clique sur les cases pour les découvrir.

Jack a dit

  • Objectif: Suivre les instructions, mais seulement si "Jack a dit".
  • Contrôles: Suis les instructions avec la souris ou les touches du clavier.

Snake

  • Objectif: Manger la nourriture sans toucher les murs ou ton corps.
  • Contrôles: Utilise les flèches pour déplacer le serpent.
